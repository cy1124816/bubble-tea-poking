# 🎨 奶茶戳戳 - 素材使用指南

## 📁 文件夹结构

```
assets/
├── images/          # 图片文件夹
│   ├── brands/     # 品牌 Logo（喜茶、CoCo等）
│   ├── empty/      # 空状态插图
│   └── bg/         # 背景图片
└── icons/          # 图标文件夹
    └── custom/     # 自定义图标
```

---

## 📸 如何准备素材

### **1. 图片格式要求**
- **格式**：PNG（推荐，支持透明背景）、JPG、SVG
- **大小**：建议每张图片 < 500KB
- **尺寸建议**：
  - Logo: 200x200px
  - 空状态插图: 300x300px
  - 背景图: 750x1334px（手机屏幕比例）

### **2. 命名规范**
用英文和数字，用 `-` 分隔，例如：
- ✅ `heytea-logo.png`（喜茶 Logo）
- ✅ `empty-state.png`（空状态图）
- ✅ `bubble-tea-bg.jpg`（奶茶背景）
- ❌ `喜茶图标.png`（不要用中文）

## 🔧 技术说明（可选阅读）

**在代码中引用方式**：
```tsx

<img src={logoImg} alt="喜茶" />
```

## 📌 常见问题

**Q: 我没有设计软件，去哪里找图片？**
A: 推荐免费资源网站：
- [Unsplash](https://unsplash.com) - 高质量照片
- [Flaticon](https://flaticon.com) - 免费图标
- [Freepik](https://freepik.com) - 插画和图标

**Q: 图片太大怎么办？**
A: 使用在线压缩工具：
- [TinyPNG](https://tinypng.com) - 图片压缩
- [Squoosh](https://squoosh.app) - Google 图片压缩工具

**Q: 可以用 emoji 吗？**
A: 可以！直接告诉我用什么 emoji 即可，例如：
"在空状态显示 🧋 emoji"
